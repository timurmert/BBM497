\section{Discussion}
Perplexity decreases as context size grows: Unigram (126.37) $>$ Bigram (88.59) $>$ Trigram (68.43). Larger n-grams capture more local context, enabling sharper next-word predictions and thus lower perplexity. Add-1 smoothing is critical given the small corpus (279 types), as most trigrams occur only once and unseen sequences would otherwise receive zero probability. The results confirm that increasing n-gram order improves model quality on this dataset.
